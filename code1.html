Estructura de una Base de Datos

Lenguaje de Consulta (SQL)

SQL (Structured Query Language) es el lenguaje utilizado para interactuar con la base de datos.

Crear una tabla

Para crear una tabla, utilizamos la sentencia CREATE TABLE:

									
										CREATE TABLE clientes (
											id INT PRIMARY KEY,
											nombre VARCHAR(50),
											dirección VARCHAR(100),
											teléfono VARCHAR(20)
											);

Esta sentencia crea una tabla llamada clientes con cuatro campos:

  • id: Número entero y clave primaria
  • nombre: Texto con longitud máxima de 50 caracteres
  • dirección: Texto con longitud máxima de 100 caracteres
  • teléfono: Texto con longitud máxima de 20 caracteres
