Estructura de una Base de Datos

  • Imagine una hoja de cálculo con filas y columnas. Cada fila es un registro. Cada columna es un campo. El nombre de la columna es el nombre del campo. La intersección de una fila y una columna es el valor del campo para ese registro.

  • Tipos de datos

    • Texto (varchar)
    • Número entero (int)
    • Número decimal (float)
    • Fecha (date)
    • Hora (time)

Tabla: Clientes

ID (int) Nombre (varchar) Dirección (varchar) Teléfono (varchar) Fecha de Nacimiento (date)
1 Juan Pérez Calle 123 555-1234 1990-02-12
2 María García Avenida 456 666-5678 1985-08-25
3 Carlos López Calle 789 777-8901 1992-01-01

  • En este ejemplo: Cada fila representa un registro (un cliente). Cada columna representa un campo (ID, Nombre, Dirección, Teléfono, Fecha de Nacimiento). El nombre de la columna es el nombre del campo. La intersección de una fila y una columna es el valor del campo para ese registro (por ejemplo, el teléfono de Juan Pérez es 555-1234).

  • Tipos de datos:

    • ID: Número entero (int)
    • Nombre: Texto (varchar)
    • Dirección: Texto (varchar)
    • Teléfono: Texto (varchar)
    • Fecha de Nacimiento: Fecha (date)
