Conceptos Básicos de Bases de Datos

Un registro es una colección de datos relacionados entre sí. Ejemplo: Un registro de un cliente contiene su nombre, dirección, teléfono, etc.

Un campo es una columna de un registro que contiene un tipo específico de dato. Ejemplo: El campo "nombre" contiene el nombre del cliente.

Una tabla es una colección de registros relacionados entre sí. Ejemplo: Una tabla de clientes contiene registros de todos los clientes.

La base de datos es el conjunto de todas las tablas y registros. Ejemplo: La base de datos contiene todas las tablas y registros de la empresa.
