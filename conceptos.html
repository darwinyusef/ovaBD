Conceptos básicos

    • Registro: Un registro es una colección de datos relacionados entre sí. Por ejemplo, un registro de un cliente podría contener su nombre, dirección, teléfono, etc.
    • Campo: Un campo es una columna de un registro que contiene un tipo específico de dato. Por ejemplo, el campo "nombre" contiene el nombre del cliente.
    • Tabla: Una tabla es una colección de registros relacionados entre sí. Cada tabla tiene un nombre y una estructura definida.
    • Base de Datos: La base de datos es el conjunto de todas las tablas y sus registros.
